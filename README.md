# Cybersecurity Framework for Rural Digital Banking

This project is a prototype web-based system designed to enhance the security of rural digital banking platforms. It demonstrates how cybersecurity mechanisms can be integrated into banking systems to protect users from financial cyber threats.

## Features

* Secure login interface for users
* Banking dashboard displaying account balance and security status
* Fraud detection system that identifies suspicious transactions
* Phishing link detection to warn users about potentially malicious websites
* Real-time security alerts for abnormal activities
* Simple and clean banking dashboard UI

## Technologies Used

* Python
* Flask
* HTML
* CSS
* Git & GitHub

## Project Objective

The objective of this project is to develop a cybersecurity framework that protects rural banking systems from common cyber threats such as phishing attacks, fraudulent transactions, and unauthorized access.
